# 🎯 Simulador de Processo Seletivo - RHTech

Este é um sistema simples de Recursos Humanos desenvolvido em Java para analisar a pretensão salarial de candidatos e validar se ela está dentro do orçamento disponível da empresa.

## 🚀 Funcionalidades

* **Cadastro de Candidato:** Coleta o nome e a vaga desejada (Estágio/Júnior).
* **Análise de Orçamento:** Avalia se a pretensão salarial bate com o teto da empresa.
* **Tomada de Decisão:** Exibe o status instantâneo de "APROVADO" ou "REJEITADO" usando estruturas condicionais.

## 🛠️ Tecnologias Utilizadas

* **Java** (Entrada de dados com `Scanner`, manipulação de textos com `String` e controle de fluxo com `if/else`).
* **Git & GitHub** (Controle de versão e documentação).
